發表於 2015-12-31

Welcome to HITCaiDog’s Blog! This is my very first post. There are some important steps listed as follows.

1.Preparation(For Windows)

node.js

git

2.Hexo

2.1 Run Git CMD,input

‘’’bash
npm install -g hexo
‘’’

2.2 Enter the directory you want to store hexo(such as E:\hexo),then input

‘’’bash
hexo init
‘’’

Hexo will automatically create all the files needed for the site in the target folder.

2.3 Change theme(not essential)

‘’’bash
git clone https://github.com/iissnan/hexo-theme-next.git themes/hexo-theme-next
‘’’
Modify the config.yml under the directory of Hexo,change the value of theme to hexo-theme-next.
Maybe useful:(update the theme file)
‘’’bash
cd theme/hexo-theme-next
git pull
‘’’

2.4

‘’’bash
hexo generate \(or hexo g),generate the local blog files.
hexo server \(or hexo s),start the local server for testing purposes.
‘’’
You can preview on http://localhost:4000
